# 📈 D&R Quantum AI - Market Terminal

Welcome to the **D&R Quantum AI Dashboard**, a professional-grade, terminal-based stock analysis tool. This software utilizes an ensemble of Machine Learning models (RandomForest, XGBoost, and GradientBoost) alongside live Yahoo Finance data to calculate long-term market trends, volatility, and technical indicators.

Created by Dan.

---

## ⚡ Features
* **Live Market Data:** Pulls up to 10 years of historical data for any valid stock ticker.
* **AI Ensemble Forecasting:** Calculates probability and confidence scores for 90, 180, and 365-day horizons.
* **Technical Gauges:** Automatically calculates MACD, RSI, Bollinger Bands, and moving averages.
* **Dynamic UI:** Runs entirely inside a custom, interactive terminal dashboard.

---

## 📥 How to Download and Install
Because this is proprietary software, the source code is kept private. You can download the compiled, ready-to-run executable directly from the Releases tab.

1. Navigate to the **[Releases](../../releases)** section on the right side of this page.
2. Download the latest `DR-Quantum-AI.zip` file.
3. Extract the folder to your desktop.
4. Double-click the `.exe` file to launch the terminal.

*Note: Windows Defender may flag the file as an "Unknown Publisher" since this is an independent build. Click **More info** -> **Run anyway** to bypass this.*
